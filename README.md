# Backup.py-Python-File-Folder-Backup-Script
Backup.py` is a Python automation tool that allows you to **safely back up files and folders** from a source directory to a destination directory.   It creates **timestamped backup folders**, logs all operations, and handles errors gracefully. This project demonstrates **file handling, folder manipulation, logging, and CLI development**.
